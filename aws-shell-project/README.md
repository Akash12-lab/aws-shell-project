# AWS Shell Scripting Project

Scripts:

1. daily_backup.sh – Backup logs daily
2. system_report.sh – CPU, RAM, Disk usage report
3. cleanup_users.sh – List inactive users
4. disk_alert.sh – Alert if disk usage >80%
5. update_notify.sh – Check for system updates

## Usage

1. SSH into your EC2 instance.
2. Make scripts executable: chmod +x *.sh
3. Run scripts: ./script_name.sh
